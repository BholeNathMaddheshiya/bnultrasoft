  • 4. CSS Selectors : CSS selectors into five categories:
    A) Simple selectors:
    1) CSS element Selector
    2) CSS id Selector
    3) CSS class Selector
    4) CSS Universal Selector
    5) CSS Grouping Selector
  • Note: id name and class name cannot start with a number!
  • B) Combinator selectors: (select elements based on a specific relationship between them)
    1) descendant selector (space)
    2) child selector (>)
    3) adjacent sibling selector (+)
    4) general sibling selector (~)
  • C) CSS Pseudo-classes: select elements based on a certain state.
    Style an element when a user mouses over it
    Style visited and unvisited links differently
    Style an element when it gets focus
    Note : use by (:)
  • D) Pseudo-elements selectors : select and style a part of an element.
    Style the first letter, or line, of an element
    Insert content before, or after, the content of an element
    Note : use by (::)
  • E) Attribute selectors : select elements based on an attribute or attribute value
    1) CSS [attribute] Selector
    2) CSS [attribute="value"] Selector
    3) CSS [attribute~="value"] Selector (Space)
    4) CSS [attribute|="value"] Selector (hyphen-)
    5) CSS [attribute^="value"] Selector (start word match)
    6) CSS [attribute$="value"] Selector (Last word Match)
    7) CSS [attribute*="value"] Selector ( selects all elements)

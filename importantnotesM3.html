Q1) Write a Python program to print all the prime numbers in the given range an interval. A prime number is a natural number greater than 1 that has no positive divisors other than 1 and itself.


Answer:

      
num=int(input("Enter the number: "))
for i in range(2, num+1):
  f=0
  for j in range(2, i): 
    if i%j==0:
      f=1
      break
  if f==0:
    print(i, end=" ")

Q4) Write a Python program to find the successor and predecessor of the largest element in an array.


Answer:

    a=[]
    for i in range(l):
      b=int(input("Enter element: "))
      a.append(b)
    print("List is:",a)
    g=a[0]
    for i in range(l):
      if g < a[i]: 
        g=a[i]
    print("Greater is:",g)
    ind=a.index(g)
    if ind+1 < l:
      print("Successor is:",a[ind+1])

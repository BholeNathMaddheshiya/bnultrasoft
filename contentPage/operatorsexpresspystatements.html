Python Operators

✬ Python Operators : Operators are used to perform operations on variables and values.

✬ Operators: These are the special symbols. Eg- + , * , /, etc.

✬ Operands: It is the value on which the operator is applied.


Types of Operators in Python:

  • 1). Arithmetic operators
  • 2). Assignment operators
  • 3). Comparison operators
  • 4). Logical operators
  • 5). Identity operators
  • 6). Membership operators
  • 7). Bitwise operators

Python Arithmetic Operators

✬ Arithmetic Operators : Arithmetic operators are used with numeric values to perform common mathematical operations.



Python Assignment Operators

✬ Assignment Operators : Assignment operators are used to assign values to variables.



Python Comparison Operators

✬ Comparison Operators : Comparison operators are used to compare two values.



Some Examples on Comparison Operators:-

Python Logical Operators

✬ Python Logical Operators : Logical operators are used to combine conditional statements.

False

True

False


Python Identity Operators

✬ Python Identity Operators : The identity operators compare the objects to determine whether they share the same memory and refer to the same object type (data type) .


Note: Python provided two identity operators.

✬ 'is' Operator

✬ 'is not' Operator



✬ Python 'is' Operator : Python provided two identity operators.

The 'is' operator evaluates to True, if both the operand objects share the same memory location. The memory location of the object can be obtained by the "id()" function .
Note:- If the "id()" of both variables is same, the "is" operator returns True.



Examples 1: on id() function:-

                       
a=5
b=10
c=15

#Printing Values
print(a)
print(b)
print(c)

# Printing IDs of a, b, and c
print("a memory ID is =>",id(a))
print("b memory ID is =>",id(b))
print("c memory ID is =>",id(c))

Output:-

5

10

15

a memory ID is => 140721431726648

b memory ID is => 140721431726808

c memory ID is => 140721431726968


Examples 2: on id() function:- Note :- When assign one value of a variable to another variable, it point the same memory location.

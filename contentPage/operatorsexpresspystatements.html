Python Operators

✬ Operators: These are the special symbols. Eg- + , * , /, etc.

Python Identity Operators

✬ Python Identity Operators : The identity operators compare the objects to determine whether they share the same memory and refer to the same object type (data type) .


Note: Python provided two identity operators.

✬ 'is' Operator

✬ 'is not' Operator



✬ Python 'is' Operator :

The 'is' operator evaluates to True, if both the operand objects share the same memory location. The memory location of the object can be obtained by the "id()" function .
Note:- If the "id()" of both variables is same, the "is" operator returns True.



Examples 1: on id() function:-

                       
a=5
b=10
c=15

#Printing Values
print(a)
print(b)
print(c)

# Printing IDs of a, b, and c
print("a memory ID is =>",id(a))
print("b memory ID is =>",id(b))
print("c memory ID is =>",id(c))

Output:-

5

10

15

a memory ID is => 140721431726648

b memory ID is => 140721431726808

c memory ID is => 140721431726968

Python Membership Operators

✬ Python Membership Operators: Python offers two membership operators to check or validate the membership of a value. It tests for membership in a sequence, such as strings, lists, or tuples.



Examples : 'in' Operator :- 

                  
a="bnultrasoft"
print("u" in a)   #returns True because "u" is present in "bnultrasoft"
print("B" in a)   #returns False because "B" is not present in "bnultrasoft" 
print("z" in a)   #returns False because "z" is not present in "bnultrasoft" 

x = ["apple", "banana"]
print("banana" in x)  #returns True because a sequence with the value "banana" is in the list

Output:-

True

False

False

True


Examples : 'not in' Operator :- 

                  
a="bnultrasoft"
print("z" not in a)  #returns True because "z" is not present in "bnultrasoft" 
print("u" not in a)  #returns False because "u" is present in "bnultrasoft" 

x = ["apple", "banana"]
print("pineapple" not in x)   #returns True because a sequence with the value "pineapple" is not in the list

Output:-

True

False

True

Python Bitwise operators

✬ Bitwise operators are used to compare (binary) numbers



Operator Name Description Syntax
& Bitwise AND Result bit 1,if both operand bits are 1;otherwise results bit 0 x & y
| Bitwise OR Result bit 1,if any of the operand bit is 1; otherwise results bit 0. x | y
^ Bitwise XOR Results bit 1,if any of the operand bit is 1 but not both, otherwise results bit 0. x ^ y
~ Bitwise NOT Inverts individual bits ~ x
>> Bitwise Right Shift The left operand’s value is moved toward right by the number of bits specified by the right operand. x >> 1
<< Bitwise Left Shift The left operand’s value is moved toward left by the number of bits specified by the right operand. x << 1

Introduction of Python

✬ Python:

✬ Python is easy to learn and use, open source and free, high level, interpreted, portable and object-oriented programming language.
✬ Python was created by Guido van Rossum, and first released on February 20, 1991.
✬ While you may know the python as a large snake, the name of the Python programming language comes from an old BBC television comedy sketch series called Monty Python's Flying Circus.
✬ It support almost all OS: Windows, macOS, Linux/UNIX, Android and a few other platforms.
✬ The files with the .py extension contain the python source code.
✬ The files with the .pyc extension contain the python compiled bytecode.


Technical Strength of Python

✬ Python is Portable:

✬ Python is portable Python is portable, it means that if we have a python code for windows. Python can be run equally on different platforms such as Windows, Linux, Mac OS etc. It enables programmers to develop the software several platform by writing a program only once.


✬ Python is Interpreted:

✬ Python is processed at the runtime by the interpreter, you do not need to compile your program before executing it.


✬ Python is Free and Open Source:

✬ In simple terms, you can freely distribute copies of this software, read its source code, make changes to it.


✬ Python is General-Purpose:

✬ It means that the Python can be used for a wide variety of development tasks including Data Science, Machine Learning, Artificial Intelligence, Software Development, Web Development etc.


Introduction to Python Interpreter and program execution


✬ Python interpreter : Python converts source code into series of bytecode in compilation stage. This bytecode cannot be understand by CPU directly. So, there is a need of mediator to do this tasks.Here interpreter comes into existence called the Python Virtual Machine.

✬ Python Virtual Machine (PVM) is a program which provides programming environment. The role of PVM is to convert the byte code instructions into machine code so the computer can execute those machine code instructions and display the output.

✬ Note : The standard implementation of python is called "CPython". Here, CPython can be defined as a both interpreter and compiler. CPython written in C and python.

✬ Step 1:- Make a file using a text editor. (Open Notepad)

✬ Step 2:- After writing the code save the file using "filename.py"

✬ Step 3:- Now, Open Command Prompt then copy the location of your python file and "Type cd" and paste the location.

Python Comments


✬ Single Line Comments

✬ Comments starts with a #, and Python will ignore them, it cannot be execute.

✬ Comments can be used to explain Python code.

Example:

print("Hello, World!") #This line says, to print Hello World.

✬ Multiline Comments

✬ Python does not really have a syntax for multiline comments. To add a multiline comment you could insert a # for each line:

Example:

#This is a comment

#written in

#more than just one line.

print("Hello, World!")

✬ But you can add comments in triple quotes (""") in your code, it treat as comments in python.

Example:

""" This is a comment
written in
more than just one line """

print("Hello, World!")

✬ Note: As long as the string is not assigned to a variable, Python will read the code, but then ignore it.

Python Variable


✬ Python Variable:

✬ Variable is the name that is used to refer memory location, it means that when you create a variable you reserve some space in memory.

✬ Python Variable we don't need to specify the type of variable.

✬ Python variable is also known as identifier.


✬ The rules for naming an identifier.

✬ A variable name must start with a letter or the underscore character.

✬ A variable name cannot start with a number.

✬ White space and special characters (!, @, #, %, etc.) are not allowed.

✬ Python is a case-sensitive programming language. So, variable names are case-sensitive (age, Age and AGE are three different variables).

✬ We cannot use Python reserve keyword has identifiers.


✬ Example of valid identifiers: a123, _n, n_9 etc.

✬ Example of Invalid identifiers: 1a, n%4, n 9 etc.

✬ The multi-word variable name can be created by the following method.

Camel Case - In the camel case, each word or abbreviation in the middle of begins with a capital letter. There is no intervention of whitespace.
For example - nameOfStudent, valueOfVariable, etc.

Pascal Case - It is the same as the Camel Case, but here the first word is also capital.
For example - NameOfStudent, etc.

Snake Case - In the snake case, Words are separated by the underscore.
For example - name_of_student, etc.

Python print() Function

✬ Python print() Function

✬ The print() function prints the specified message to the screen, or other standard output device.

✬ The message can be a string, or any other object, the object will be converted into a string before written to the screen.

The full syntax of print() is:

    
print(object(s), sep=' ', end='\n', file=sys.stdout, flush=False)
Parameter Description
object(s) Any object, and as many as you like. Will be converted to string before printed.
sep='separator' Optional. Specify how to separate the objects, if there is more than one. Default is ' '
end='end' Optional. Specify what to print at the end. Default is '\n' (line feed)
file Optional. An object with a write method. Default is sys.stdout
flush Optional. A Boolean, specifying if the output is flushed (True) or buffered (False). Default is False.

✬ Python String Indexing: In Python, individual characters of a String can be accessed by using the method of Indexing.

a) The indexing of the Python strings starts from 0.

b) Indexing allows negative address references to access characters from the back of the String,
e.g. -1 refers to the last character, -2 refers to the second last character, and so on.

Note: 1 While accessing an index out of the range will cause an IndexError.
IndexError: string index out of range.

Note: 2 Only Integers are allowed to be passed as an index, float or other types that will cause a TypeError.
TypeError: string indices must be integers, not 'float'.


✬String Slicing in Python: Python slicing can be done in two ways:

1) Using the array slicing [:: ] method
2) Using a slice() method


✬ 1) Using the List/array slicing [ :: ] method :-

a) In Python, the slice operator [n] is used to access the individual characters of the string. where n is number of index.

Example

                            
s="BNUltrasoft"
print(s[2])

Output: U

b) Slicing in a String is done by using a Slicing operator, i.e., a colon (:).
One thing to keep in mind while using this method is that the string returned after slicing includes the character at the start index but not the character at the last index.

Example

                            
s="BNUltrasoft"
print(s[2:7])

Output: Ultra

⇛ The find() method finds the first occurrence of the specified value.

x = txt.find("e", 5, 10)

print(x)

Output :

8

-1

✬ The strip() method removes any leading, and trailing whitespaces.

✬ Leading means at the beginning of the string, trailing means at the end.

✬ You can specify which character(s) to remove, if not, any whitespaces will be removed.

Example 1: Remove spaces at the beginning and at the end of the string:

                    
txt = "     banana     "

x = txt.strip()

print("of all fruits", x, "is my favorite")

Output :-

of all fruits banana is my favorite


Example 2: Remove the leading and trailing characters:

                    
txt = ",,,,,rrttgg.....banana....rrr"

x = txt.strip(",.grt")

print(x)

Output :-

banana


✬ The lstrip() method removes any leading characters (space is the default leading character to remove).

Example 1 : Remove spaces to the left of the string:

                    
txt = "     banana     "

x = txt.lstrip()

print("of all fruits", x, "is my favorite")

Output :- 

of all fruits banana      is my favorite

Example 2 : Remove the leading characters:

                    
txt = ",,,,,ssaaww.....banana"

x = txt.lstrip(",.asw")

print(x)

Output :-

banana


✬ The rstrip() method removes any trailing characters (characters at the end a string), space is the default trailing character to remove.

Example 1: Remove any white spaces at the end of the string:

                    
txt = "     banana     "

x = txt.rstrip()

print("of all fruits", x, "is my favorite")

Output :- 

 of all fruits      banana is my favorite

Example 2: Remove the trailing characters if they are commas, periods, s, q, or w:

                    
txt = "banana,,,,,ssqqqww....."

x = txt.rstrip(",.qsw")

print(x)

Output :-

banana

✬ The partition() method searches for a specified string, and splits the string into a tuple containing three elements.

  • ⇛ The first element contains the part before the specified string. (everything before the "match")
  • ⇛ The second element contains the specified string. (the "match")
  • ⇛ The third element contains the part after the string. (everything after the "match")

Example 1:

                    
txt = "I could to eat bananas all day"

x = txt.partition("eat")

print(x)

Output :-

('I could to ', 'eat', ' bananas all day')

Note: This method searches for the first occurrence of the specified string.

Example 2:

                    
txt = "I could to eat bananas all day"

x = txt.partition("apple")

print(x)

Output:-

('I could to eat bananas all day', '', '')

  • ✬ If the specified value is not found , the partition() method returns a tuple containing:
  • ⇛ 1 - the whole string
  • ⇛ 2 - an empty string
  • ⇛ 3 - an empty string

Python Strings encode() method

✬ Python String encode() converts a string value into a collection of bytes, using an encoding scheme specified by the user.

✬ Note :- Python uses utf-8 as its encoding by default.





Python Strings decode() method

✬ This works opposite to the encode. It accepts the encoding of the encoding string to decode it and returns the original string.


Example of encode and decode : ascii to ascii

                    
# defining a new string
example = "BNUltrasoft"

# Encoding the input string and storing it in a new variable
encodedString = example.encode(encoding='utf8')

# Displaying the encoded string
print ("Input string in the encoded format: ")
print (encodedString)

# Displaying the decoded string
print ("Input string in the decoded format: ")
decodedString = encodedString.decode('utf8', 'strict')
print (decodedString)

Output :-

Input string in the encoded format:

b'BNUltrasoft'

Input string in the decoded format:

BNUltrasoft


Example of real encode :- 

                    
txt = "My name is St√•le"

print(txt.encode(encoding="ascii",errors="backslashreplace"))
print(txt.encode(encoding="ascii",errors="ignore"))
print(txt.encode(encoding="ascii",errors="namereplace"))
print(txt.encode(encoding="ascii",errors="replace"))
print(txt.encode(encoding="ascii",errors="xmlcharrefreplace"))
print(txt.encode(encoding="ascii",errors="strict"))

Output :-

b'My name is St\\xe5le'

b'My name is Stle'

b'My name is St\\N{LATIN SMALL LETTER A WITH RING ABOVE}le'

b'My name is St?le'

b'My name is St &# 229; le'

Python pow() Function

✬The pow() function returns the value of x to the power of y (x y).

✬ If a third parameter is present, it returns x to the power of y, modulus z.

Example :- Return the value of 2 to the power of 3 (same as 2 * 2 * 2):


x = pow(2, 3)

Output :-

8


Example 2: Return the value of 2 to the power of 3, modulus 5 (same as (2 * 2 * 2) % 5):

3

Python math.sqrt() Method

✬ math.sqrt() :- The math.sqrt() method returns the square root of a number.

⇛ Note : The number must be greater than or equal to 0.


Example :

                    
import math

print (math.sqrt(9))
print (math.sqrt(25))

Output :-

3.0

5.0



Example 2:

                    
import math

print (math.sqrt(-5)) #ValueError: math domain error
print (math.sqrt("A")) #TypeError: must be real number, not str
print (math.sqrt(A))  #NameError: name 'A' is not defined

Note :-

⇛ If the number is less than 0, it returns a ValueError

⇛ If the value is not a number, it returns a TypeError

⇛ If the value is not defined, it returns a NameError

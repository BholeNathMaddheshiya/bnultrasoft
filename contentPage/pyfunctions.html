✬ The partition() method searches for a specified string, and splits the string into a tuple containing three elements.

  • ⇛ The first element contains the part before the specified string. (everything before the "match")
  • ⇛ The second element contains the specified string. (the "match")
  • ⇛ The third element contains the part after the string. (everything after the "match")

Example 1:

                    
txt = "I could to eat bananas all day"

x = txt.partition("eat")

print(x)

Output :-

('I could to ', 'eat', ' bananas all day')

Note: This method searches for the first occurrence of the specified string.

Example 2:

                    
txt = "I could to eat bananas all day"

x = txt.partition("apple")

print(x)

Output:-

('I could to eat bananas all day', '', '')

  • ✬ If the specified value is not found , the partition() method returns a tuple containing:
  • ⇛ 1 - the whole string
  • ⇛ 2 - an empty string
  • ⇛ 3 - an empty string

Python Strings encode() method

✬ Python String encode() converts a string value into a collection of bytes, using an encoding scheme specified by the user.

✬ Note :- Python uses utf-8 as its encoding by default.





Python Strings decode() method

✬ This works opposite to the encode. It accepts the encoding of the encoding string to decode it and returns the original string.


Example of encode and decode : ascii to ascii

                    
# defining a new string
example = "BNUltrasoft"

# Encoding the input string and storing it in a new variable
encodedString = example.encode(encoding='utf8')

# Displaying the encoded string
print ("Input string in the encoded format: ")
print (encodedString)

# Displaying the decoded string
print ("Input string in the decoded format: ")
decodedString = encodedString.decode('utf8', 'strict')
print (decodedString)

Output :-

Input string in the encoded format:

b'BNUltrasoft'

Input string in the decoded format:

BNUltrasoft


Example of real encode :- 

                    
txt = "My name is St√•le"

print(txt.encode(encoding="ascii",errors="backslashreplace"))

print(txt.encode(encoding="ascii",errors="namereplace"))
print(txt.encode(encoding="ascii",errors="replace"))
print(txt.encode(encoding="ascii",errors="xmlcharrefreplace"))
print(txt.encode(encoding="ascii",errors="strict"))

Output :-

✬ Parameters or Arguments?

A parameter is the variable listed inside the parentheses in the function definition.

An argument is the value that is sent to the function when it is called.


✬ Number of Arguments

Example :


def my_function(x,y):
    print("I Like to eat",x,y)

my_function("Mango","Banana")

Output :-

I Like to eat Mango Banana


✬ Number of Arguments

Note:- By default, a function must be called with the correct number of arguments. Meaning that if your function expects 2 arguments, you have to call the function with 2 arguments, not more, and not less.

Example :


def my_function(x,y,z):
    print("I Like to eat",x,y)

my_function("Mango","Banana")

Output :-

TypeError: my_function() missing 1 required.


Arbitrary Arguments, *args

If you do not know how many arguments that will be passed into your function, add a * before the parameter name in the function definition.

Example :


def my_function(*x):
    print("I like: ",x[2])

my_function("Mango","Banana","Apple")

Output :-

I like: Apple


✬ Keyword Arguments

⇛ You can also send arguments with the key = value syntax.

⇛ This way the order of the arguments does not matter.

Example :


def my_function(x,y,z):
    print("I like: ",z)

my_function(x="Mango",z="Banana",y="Apple")

Output :-

I like: Banana

Global and Local Variables in Python


✬ Local Variables: Local variables in Python are those which are initialized inside a function and belong only to that particular function. It cannot be accessed anywhere outside the function.

Example:Creating and accessing local variables:

                    
def f():
    
    # local variable
    s = "Python"
    print(s)

f()
print(s) #Trying to access the local variable.

Output :-

Python

NameError: name 's' is not defined



✬ Python Global variables are those which are not defined inside any function and have a global scope.

Example: Creating and accessing global variables: 

                    
# This function uses global variable s
def f():
    print("Inside Function:", s)

# Global scope
s = "I love Python"
f()
print("Outside Function:", s)

Output :-

Inside Function: I love Python

Outside Function: I love Python

Web Site Development using W3.CSS Framework

W3.CSS Introduction

✬ : W3.CSS is a modern, responsive, mobile first CSS framework.

✬ : W3.CSS provides equality for all browsers: Chrome. Firefox. Edge. IE. Safari. Opera.

✬ : W3.CSS provides equality for all devices: Desktop. Laptop. Tablet. Mobile.

✬ : W3.CSS is standard CSS only (No jQuery or JavaScript library).

✬ : W3.CSS is free to use. No license is necessary.


How To Use : Method 1 (Online Method)

To use W3.CSS in your web site, just add a link in <head> in your web pages but it required internet connection while running the website.


The Link is :


<meta name="viewport" content="width=device-width, initial-scale=1">
      
<link  rel="stylesheet"  href="https://www.w3schools.com/w3css/4/w3.css">

How To Use : Method 2 (Offline Method)

✬ To use offline method we have to download the css file and add file in your website.




<meta name="viewport" content="width=device-width, initial-scale=1">

<link  rel="stylesheet"  href="w3.css">

W3.CSS Fonts

★ Verdana is the default font used in W3.CSS.

★ Segoe UI is the default font for headings.


Using Font Classes With W3.CSS you can use 4 different built-in font classes:

1) w3-serif : 


  <p class="w3-serif">

2) w3-sans-serif : 


  <p class="w3-sans-serif">

3) w3-monospace : 


  <p class="w3-monospace">

4) w3-cursive : 


  <p class="w3-cursive">


Note :

★ Arial is ranked #1 in use on the Web.

Responsive Table

The w3-responsive class creates a responsive table. The table will then scroll horizontally on small screens. When viewing on large screens, there is no difference. 



<!DOCTYPE html>
<html>
<head>
<title>W3.CSS</title>
<meta name="viewport" content="width=device-width, initial-scale=1">
<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">
</head>
<body>
                        
<div class="w3-responsive">
<table class="w3-table-all">
  ... table content ...
</table>
</div> 
                        
</body>
</html>

Output (Result) of above:

A responsive table will display a horizontal scroll bar if the screen is too small to display the full content.

Resize the screen to see the effect.

W3.CSS Grid

W3.CSS supports a 12 column responsive fluid grid.


★ s1 - s12 (Columns for small screens (typical smart phones))

★ m1 - m12 (Columns for medium screens (typical tablets))

★ l1 - l12 (Columns for large screens (typical laptops and desktops))




Two Equal Columns

Two equal-width columns (50%/50%) on all screen sizes:



<div class="w3-row">
 <div class="w3-col s6 w3-green w3-center"><p>s6</p></div>
 <div class="w3-col s6 w3-dark-grey w3-center"><p>s6</p></div>
</div>

Output (Result) of above:




Three Unequal Columns

Three various-width columns (25%/50%/25%) on tablets, laptops and desktops. On mobile phones, the columns will automatically stack (100% width):


<div class="w3-row">
 <div class="w3-col m3 w3-green w3-center"><p>m3</p></div>
 <div class="w3-col m6 w3-dark-grey w3-center"><p>m6</p></div>
 <div class="w3-col m3 w3-red w3-center"><p>m3</p></div>
</div>

Output (Result) of above:




Mixed: Mobile and Laptops

You can combine classes to create a dynamic and flexible layout. This example will produce a two column layout with a 83.34%/16.66% (l10, l2) split on large screens and a 50%/50% (s6, s6) split on small screens:


<div class="w3-row">
 <div class="w3-col l10 s6 w3-pink w3-center"><p>l10 s6</p></div>
 <div class="w3-col l2 s6 w3-dark-grey w3-center"><p>l2 s6</p></div>
</div>
